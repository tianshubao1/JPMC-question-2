# Bonus Question: HMC with Invertible Flows & Differentiable Resampling

This folder contains the implementation for **Bonus Question 1**, which explores advanced Bayesian inference methods by combining Particle Filters with MCMC strategies (PMMH) and Differentiable Programming (HMC).

---

## 1. Requirements

Ensure you have the following installed. **Note:** `tensorflow-probability` and `tf-keras` are strictly required for Part (b).

```txt
numpy
matplotlib
scipy
tensorflow
tensorflow-probability
tf-keras
```

```
.
├── bonus_part_a_pmmh.py       # Part (a): PMMH with Li(17) Invertible Flow
├── bonus_part_b_hmc.py        # Part (b): HMC with Differentiable PF (Sinkhorn OT)
```
